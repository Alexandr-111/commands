## Git Bash для Windows основные команды
 + **pwd**  *выводит путь к текущей директории*
 + **cd**  *перейти в указанную директорию*
`$ cd ~` *перейти к домашней директории*
 + **ls**  *вывести содержимое директории*
 + **ls -a**  *напечатать содержимое директории со скрытыми файлами* 
 + **.**  *обратиться к текущей директории (одна точка)*
 +  **..**  *обратиться к родительской директории (на один уровень выше)*
 +  **touch** *создать файл*
 +  **mkdir** *создать директорию (папку)*
 + $ **mkdir -p** *создать структуру директорий*
  `$ mkdir -p dir1/dir2/dir3`
 + **cp** *копирование файла*
 `$ cp index.html src/` *файл и папка куда его скопировать*
 +  **mv** *перемещение файлов и папок*
   `$ mv table.txt very-important-files`
 + **cat** *чтение файла*
 `$ cat mfile.txt`
 + **rm** *удаление файла*
 `$ rm example.txt` 
 + **rmdir** *удаление папки
 + **rm -r** *удаление папок и файлов внутри директории и самой директории* 
 + && *последовательность команд*
 `$ mkdir second-project && cd second-project && touch index.html` 
 + **git version** *версия Git*
 + **git config --global** глобальные настройки
 + **$ git config --list** вывести содержимое файла конфигурации 
 + **git init** *сделать папку репозиторием*
 + **rm -rf .git** *"разгитить" папку*
 + **git status** *проверить состояние репозитория*
 + **git add** *подготовить файлы к сохранению* 
`$ git add readme.txt`
 + **git commit** *выполнить коммит*
 `git commit -m "Комментарий к коммиту."` *коммит с комментарием*
 + **git log** *просмотреть историю коммитов*
 + **clip <** *скопировать содержание файла в буфер обмена
 `clip < ~/.ssh/id_ed25519.pub`
 + **ssh-keygen** *генерировать ssh-ключи*
 + **git remote add** *привязать удалённый репозиторий к локальному*
`$ git remote add origin git@github.com:%ИМЯ_АККАУНТА%/first-project.git`
 + **git remote -v**  *проверить, что репозитории связаны* 
 + **git push** *отправить изменения на удалённый репозиторий*
 + **git clone** *клонировать репозиторий*
 `$ git clone https://github.com/yandex-praktikum/git-clone-lesson`
 + **git help** *вывести список основных команд Git*
 + **git help -a** *вывести список всех команд Git*
